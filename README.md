## Project Description

This project analyzes and documents the requirements, workflow, and testing strategy for a **Digital Parking Permit System** for a university environment. The system allows students and staff to register vehicles, request parking permits, check available zones, and report issues. Security staff can verify permits and log violations.

### Task 2 – Workflow Diagram

### Task 3 – Test Case Table
6 test cases covering:
- 2 System Tests
- 2 Acceptance Tests

### Task 4 – Defect Analysis
Defect: "A valid parking permit is shown as expired during security verification."
